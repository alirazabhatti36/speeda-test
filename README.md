# Speeda Test

Speeda Test is a full-stack web application for internet speed testing and website performance analysis.

## Features

- Internet speed test (download, upload, ping)
- Website performance test with basic and detailed modes
- Responsive frontend UI with React + Vite
- Backend API with Express

## Project Structure

- `backend/` API server
- `frontend/` React application
- `.github/` issue and PR templates

## Getting Started

### 1. Install dependencies

```bash
npm run install:all
```

### 2. Run development servers

```bash
npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:5000

### 3. Build frontend

```bash
npm run build
```

## Scripts

- `npm run dev` start backend + frontend in dev mode
- `npm run build` build frontend
- `npm run preview` preview production build
