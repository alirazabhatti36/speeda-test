## 🚀 Features

- ⚡ **Real Speed Test** — Accurate download, upload, and ping measurements
- 🧠 **AI-Powered Smart Termination** — 1.8-4.4x faster than traditional tools
- 🌐 **Website Speed Tester** — Test any website's performance
- 📱 **Fully Responsive** — Works on all devices
- 🔒 **Privacy First** — No data collection, no storage

## 🧠 AI Smart Termination

Traditional speed tests run for a fixed duration (10+ seconds). Speeda Test uses AI to:

1. **Predict** final speed from initial samples
2. **Detect** when speed stabilizes
3. **Terminate** early to save data and time
4. **Result:** 1.8-4.4x faster with same accuracy

## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/alirazabhatti36/speeda-test.git

# Install all dependencies
npm run install:all

# Run development server
npm run dev
```
